Profundizacion: Modelo OSI y TCP/IP

El modelo OSI de la ISO, es un estandar que forma un conjunto de 7 capas en donde cada capa tiene el fin de organizar como se mueven los datos dentro de las redes de datos, hare enfasis en las siguientes capas:

  • Capa fisica: Maneja la transmision de datos mediante medios fisicos como cables o tambien señales
  • Capa de transporte: Utiliza protocolos como TCP y UDP para asegurar la integridad de los datos entre dispositivos

    TCP/IP

    A diferencia del modelo teorico OSI que ayuda mayormente a entender como se mueven los datos dentro de una red, el modelo TCP/IP es un modelo practico usado en el dia a dia.

    Donde TCP(protocolo de control de transmision) Es un transporte de datos y IP(protocolo de internet) Son las direcciones a donde se envian esos datos, cuenta con 4 capas, Hare enfasis en las siguientes

  • Capa de enlace: Es el acceso fisico a la red, como el puerto RJ45 LAN de un router
  • Capa de transporte: Aqui se pueden configurar puertos para asociarlos a APPs de datos
